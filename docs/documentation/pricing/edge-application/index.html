Pricing Example

If you have contracted Enterprise support and have 100 domains and 60 edge applications in Real-Time Manager, the value will be calculated as follows:

Domains

The first 50 domains are included as contracted support. The next 50 domains have an add value of 5 USD each, so 50 x 5 USD = 250 USD.

Edge Application

The first 50 edge applications are included as contracted support. The next 10 edge applications have the additional value of 5 USD each, so 10 x 5 USD = 50 USD.

Valor total

The total amount to be paid in the month will be 300 USD, according to the simulation.
