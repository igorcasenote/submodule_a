Pricing Example

If your Application Acceleration enabled application generated within 30 months 30 TB of data trafficked in Brazil, the value will be calculated as follows:

The first 10 TB trafficked are priced at $ 0.040 per GB, so $ 10,000 x $ 0.040 = $ 400.

The next 20 TB trafficked are priced at $ 0.035 per GB, so $ 20,000 x $ 0.035 = $ 700.

In this example, the value for the month will be $ 1,100.
