Pricing Example

If your applications invoke an edge function in the United States 5 million times in a month, and each invocation runs within 30 milliseconds, you will be billed as follows.

Compute time: (5,000,000 summons x 0.030 seconds / 1,000) x 0.049 USD = 7.35 USD.

Invocations: (5,000,000 / 1 million) x 0.60 USD = 3 USD.

The total that will be charged, in this example, is $ 10.35.
